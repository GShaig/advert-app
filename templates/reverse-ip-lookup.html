What is an IP Address?




What is an IP address and Reverse IP Lookup?

An IP (Internet Protocol) address is a network address for your computer or for the server where your website is hosted in. It is like a domain name address but with a little difference.

IP addresses can change or can be reassigned in some time. IP addresses are like identity cards for technological communications between computers. The basic format for IPv4 IP addresses is like 123.234.123.012.

When you check one IP address information, you actually check the information about the computer or the server the IP address attached to. You can also see the information about the organization who owns that computer or server.

Typically for IPv4 addresses, 127.0.0.1 represents local host and 0.0.0.0 represents all the IP addresses. You can learn your IP address by clicking on My IP Address button which located on top right corner of this website.

IPv6 IP addresses are much longer than IPv4 ones and they are typically combination of numbers and letters. The format for an IPv6 type IP address is like 2001:0db8:85a3:0000:0000:1a2b:3456:7890. An IPv6 address is represented with a 128 bits value. That is why IPv6 has a wider address coverage than IPv4 which uses 32-bit values to represent IP addresses.

You should be aware of that the provided IP addresses may not be accurate since internet users can use a VPN (Virtual Private Network) or a reverse proxy server that can change their IP addresses to show a totally different location. This is usually done to bypass some website and service blocking in certain countries.

Please note that the Reverse IP Lookup tool on this website is not a DNS Lookup and does not show the DNS records associated with a specific domain.



What is a port number?

If an IP address was a street address, the port number could be a unit number. There is a colon (':') mark between the IP address and the port number.

For example, to show the port 5000 on local host, we write 127.0.0.1:5000. Port numbers are used to run multiple servers on the same host machine.

For instance, you can run one application on port number 3000 and the other application on port number 5000 on your local computer. If you wish to communicate with the application running on port 3000, just put 127.0.0.1:3000 on your browser address bar.





Can your IP address change over time?

Simple answer to this question is YES. Your IP address can change over time. But this depends on many factors.

Firstly, it depends on your internet service provider. Your ISP can assign a static IP address to your modem or connection. In this case, your IP address will be fixed. But most internet service providers use dynamic IP addresses for many reasons.

If you have a website and you wonder if your website IP address changes over time or not, it is again depends on your domain name provider or your hosting plan provider. Hosting companies usually use two types of IP addresses: internal IP addresses and external IP addresses.

Change of internal IP addresses is entirely depends on how long you use that specific service or if you refresh your resources frequently or not. Normally when you refresh a service, let it be a load balancer, a container or a database service, this process changes its IP address.

But changing external IP addresses is a process most service providers do on purpose, due to security concerns. Additionally, most service providers give you a static IP address if you wish to use, such as AWS (Amazon Web Services) Elastic IP service.



The technology behind an IP address.

IPv4 and IPv6 type IP addresses are just human readable copies of the Internet Protocol. Normally, IP addresses are in binary format in order for the internet devices to be able to communicate with each other. For example, the binary version of 123.234.123.234 is 01111011.11101010.01111011.11101010.

An IP address identifies three main parameters: the name of the host and its network interface, the location of the host, and the way how to establish a connection with the host.

The need for IPv6 type IP addresses was brilliant clear when the authorities saw the drastic expansion of the Internet space back in 1995. An unprecedented increase in the number of ISPs and devices on the internet obviously created a high demand for IP addresses. In that point, IPv6 was introduced to the Internet world. The numerical value used for an IPv6 type IP address is 128 bits value and obviously IPv6 can accommodate far a greater number of IP addresses than IPv4 can.

All IP addresses are managed by the Internet Assigned Numbers Authority. And there are five regional Internet registries which operate in five different continents (North America, South America, Eurasia, Africa, and Oceania). These RIRs manage the assignment of IP addresses to Internet service providers in those areas.
