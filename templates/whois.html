How to use WHOIS Lookup Tool?

To use WHOIS Lookup Tool, just enter your desired domain name to the search box provided above and click on the Check button.

You will get the WHOIS lookup results instantly! The information will include the owner of the domain name, the contact details, on which date the domain was registered, the expiry date of the domain name etc.

Sometimes you will see that some information on your WHOIS lookup results are hidden. That is because the domain owner has purchased and enabled domain privacy for that specific domain name.

Please note that you can perform a WHOIS lookup also with an IP address instead of a domain name. The provided information will be a little different and the technical properties of the IP address will also be included in WHOIS lookup results.

If the domain name you are searching for has never been registered before, a green warning will show up with the text Domain name is not registered, yet. There will also be a green Register button for you in case you wish to buy the domain name.

This WHOIS lookup tool works for all domain extensions and country codes. If you experience issues for a specific domain extension, please let us know.

If you have specific requests or suggestions for the domain owner, please use the contact details of the domain owner, not the domain registrar. If you think the information provided in this tool is inaccurate, please use ICANN Whois Inaccuracy Complaint Form link provided on the check results to report the issue.

You can use this tool to check the information about your own domain name as well. Check if the information about your domain on WHOIS database is up-to-date or not. If the information is not accurate or recent, you can just contact your domain registrar to correct or update your information on the database.

There is no limit how many searches each user perform with the WHOIS lookup tool. If you face any problem related to this tool or if the tool does not work, please report us the situation on info@adverttools.com email address. Thank you!

Do you need more information on what is WHOIS lookup and how does it work? Click on What is WHOIS Lookup and Use Cases of WHOIS Lookup links to get the necessary information.



Top eight interested facts about WHOIS lookup.

1. It is very easy to put WHOIS lookup on your website. You just need to send API calls in your preferred programming language such as Python, Java, PHP, JavaScript etc. Then receive response, parse and render it on your web page.

2. Most WHOIS APIs accept querying also with an IP address. So, you can both send a domain name or an IP address to request the information about it. But provided information will be a little different when you are querying to get the information about an IP address.

3. Usually WHOIS APIs return an error when you request the information about an unregistered domain name. But some APIs return Domain Not Found response as string. So, you need to write an exception to handle this situation.

4. The registrations of com, org and net domain extensions is handled by Internet Corporation for Assigned Names and Numbers (ICANN).

5. Have you checked the domain status information on your WHOIS lookups before? If you checked it, you would know in some cases domain status shows clientDeleteProhibited, clientTransferProhibited or clientUpdateProhibited. If these domain statutes are enabled, it means that the domain is protected against domain theft. Many domain name registrars provide this as a service to protect their clients from losing their domain names. This service is usually a premium service and incur charges.

6. IANA ID shown on WHOIS lookup results means Internet Assigned Numbers Authority and is like a registration ID for domain registrars.

7. If you are familiar with the common name servers of hosting companies, you can find out the hosting provider of a domain by simply checking its name servers on WHOIS lookup results.

8. Even if the data about a particular domain name is hidden on WHOIS lookup, the email address of the domain registrar is always available and shown on the check results. This is due to the security measure which allows the internet users report the domain name in case of abuse or other unwanted situations.



WHOIS Lookup FAQs


How can I know who owns a domain?

Please use our WHOIS Lookup tool. Just enter the domain name in the search box and click on "Check".


Is WHOIS Lookup free of charge?

Yes, WHOIS Lookup is completely free. You can check as many domain names as you wish. The sky is the limit!


What to do if some information about a particular domain name are not shown?

Some people or companies prefer to hide their information from WHOIS lookup results such as name, address or contact details.

There is no way to find out the hidden information if domain privacy is enabled for a specific domain name.


What if the searched domain name is not registered yet?

If the domain name searched for is not registered yet, a green Register button will be shown. You can click the button and buy the domain name.
