How to use Reverse IP Lookup Tool?

Copy the IP address you would like to know about and paste it to the search box provided above. And then click on the Check button.

Then wait for a second before you see the information about the IP address you entered. The information includes registered organization (or owner) of an IP address, date of registration, location, contact details etc.

Please note that this IP lookup tool supports only IPv4 type IP address searches. We are planning to add IPv6 support feature in the coming months.

In contrast to WHOIS lookup, IP lookup results include additional information about the technical properties of the searched IP address such as net range, classless inter-domain routing, net name, net handle, net type etc.

IP lookups for some IP addresses will return hidden or very limited information. This is because the IP address you searched for probably belongs to a governmental institution or a strategic governmental agency from one of the countries. The information is restricted because of security concerns.

Typical IPv4 type IP address format is like 123.456.789.012. You can learn your IP address simply by clicking on My IP Address button on top left corner of this page.

Did you check your own IP address and see that the provided information is incorrect? No worries! Please contact the relevant IP address authority shown on your check results and ask to update your IP address. There is usually a link to a complaint form to fill and submit for any request. An information update can take several weeks to be shown on the lookup results.

If you got a cyber attack or someone tried to hack you from a known IP address, you should report this IP address abuse to the relevant authorities. In order to do that please use the abuse contact details shown on the relevant IP lookup result.

If you need more detailed information about what is a reverse IP lookup, please click on What is Reverse IP Lookup link.



Some important information you will get from the Reverse IP Lookup tool.

Connected domain name. - If there is a connected domain name to the IP address, the domain will be shown on the IP lookup results.

NetRange - This information shows how many host devices can exist in the network the IP address belongs to.

Classless Inter-Domain Routing (CIDR) - CIDR is the notation for the IP address. There are three types of IP addresses: Class A, Class B, and Class C

NetType - The company the IP address is allocated to.

RegDate - Registration date of the IP address.

Updated - The date when the IP address was updated lastly.

org-name - Name of the organization hires the IP address temporarily.

person - The name of the responsible person from the tenant organization.

Please note that if the IP address you are looking for is a reserved IP address for different purposes, the information related to the IP address will be limited.

If you are searching for an IP address connected with a domain and a web server, the results will be similar to Domain Lookup (WHOIS Lookup). But if you enter an IP address associated with a device like internet modem or router which assigned by an internet service provider (ISP), the results will include more technical details that can be understood only by a technical person such as a network engineer or IT helpdesk specialist.

If you enter IPv6 type IP address to this IP lookup tool, the tool will recognize the IPv6 IP address as a domain name due to the text content included in IPv6 IP format. As a result, the tool will tell that the domain name is not registered yet and no data available for your search.



IP Lookup FAQs


How do I find out the location of a Domain or an IP address?

Please use our Domain Location tool or Reverse IP Lookup tool to find out the location.


Does reverse IP lookup tool accept IPV6 IP addresses?

Unfortunately, for now the reverse IP lookup tool does not accept IPv6 IP addresses. We continuously work on the tool and maybe in the near future we will be able to add the IPv6 feature.


Can I include the port number in an IP lookup?

No, please do not include the port number. Port numbers serve a different purpose and it does not make sense to include them in IP lookups.


What information is provided during IP lookup?

Reverse IP Lookup tools provides information about the net range, classless inter-domain routing (CIDR), the organization the IP address is allocated to, the company hired the IP address, contact details and the location of an IP address.

Please note that some of these information may not be available depending on the IP address you searched for.
