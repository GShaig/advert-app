How to use Domain Location Tool?

It is very easy! Just type in your interested domain name to the search box provided above and click on the Check button.

The results will be shown instantly! You will see the domain name registered country, state, city, postcode, and (if available) the exact address where the domain name is registered in.

In some cases, the city, postcode, or address information about the domain name will be hidden. This happens mostly for the open address information. Some organizations and governmental agencies hide their address information from domain lookups due to the security concerns.

Please note that the address information shown on the domain location lookup tool result may not be the exact address of the domain owner or the organization the domain name registered for. It is rather the address of the server where the domain name is hosted, in most cases.

The address information shown on this tool may be the old address of a person or an organization. For more recent address information, you should always refer to the domain website.

Some locations shown on the Domain Location lookup will be available as an exact open address, but in the other ones only country, city and state information will be accessible. Some parts in some domains are hidden depending on the domain owner preferences.

Did you check your domain location and see that the address information is inaccurate or outdated? Please contact your domain registrar to correct or update your address information on WHOIS database.

Please be aware of that this domain location tool uses alpha-2 country codes to represent the country name in the provided location information. If you so not know what is alpha-2 country code or you are not familiar with its terminology, please visit this page Alpha 2 Country Codes. Please note that alpha-2 country codes are the same with the domain extension country codes. For example, .us is the country domain extension for the United States and US is the alpha-2 country code again for the United States.

If you think that the location information provided in this tool is not accurate or irrelevant, please report the case to us on info@adverttools.com email address. Thank you for your contributions.

If you need more detailed information about domain location lookups, please do not hesitate to visit our Why to Check Domain Location page.



Important points to take into account when checking a domain location.

- If you wish to learn the address of the company who owns the domain name, the domain location lookup tool may not be a proper solution for you. Because in some cases this tool shows the address of the data center where the website is hosted or the headquarter of the domain registrar.

- Occasionally the address information for a specific domain name will be hidden. This is because of a privacy setting enabled by the domain owner. But the country and state information will always be visible for you.

- When you enter your domain name in this tool, the address shown on the screen might be different from your actual address. You may see the address of your domain registrar or the address of your hosting service provider.

- If you enter a domain name without a domain extension on this tool, you will see the address information for the same domain with the .com domain extension. To see the address information for the domains with a domain extension other than .com, please explicitly specify the domain extension.

- This tool does not support location lookup for subdomains. We understand that the subdomain of a domain name can represent a different server with a different location. But this tool supports the location lookups only for root domains for now.

- If the requested domain name is not registered yet, a yellow warning will be shown on the screen with the note Domain is not registered.. There will also be a BUY button next to the warning in order to make it easier for you to purchase the domain name in case you wish to buy it.



Domain Location FAQs


How do I find out the location of a Domain or an IP address?

Please use our Domain Location tool or Reverse IP Lookup tool to find out the location.


What information the domain location lookup provides?

The domain location lookup tool provides country, state, city, postcode, and if available, the exact address information for the domain name.


What does mean Redacted for Privacy?

Redacted for Privacy phrase means the domain owner or company have got domain privacy to hide the location information from lookup results.


Is the location information 100% accurate?

Please note that the location information provided in this tool is not guaranteed to be 100% accurate. This situation can have many different reasons.

The provided address information can belong to the domain registrar instead of the domain owner or simply it can be the address of the servers where the domain is hosted.
